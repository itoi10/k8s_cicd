Go Echo API

使用方法: エンドポイントをCURLなどでリクエスト

ユーザー登録 POST /signup

Request curl -X POST (IPアドレス):8080/signup -H "Content-Type: application/json" -d '{"name": "user1", "password": "password"}'
Resopnse {"id":1,"name":"user1","password":""}

ログイン POST /login

Request curl -X POST (IPアドレス):8080/login -H "Content-Type: application/json" -d '{"name": "user1", "password": "password"}'
Resopnse {"token":"(JWTトークン)"}

todo登録 POST /api/todos

Request curl -X POST (IPアドレス):8080/api/todos -H "Authorization: Bearer (JWTトークン)" -H "Content-Type: application/json" -d '{"name": "todo1"}'
Resopnse {"uid":1,"id":1,"name":"todo1","completed":false}

todo一覧取得 GET /api/todos

Request curl -X GET (IPアドレス):8080/api/todos -H "Authorization: Bearer (JWTトークン)"
Resopnse [{"uid":1,"id":1,"name":"todo1","completed":false}]

todo更新 PUT /api/todos/:id/completed

Request curl -X PUT (IPアドレス):8080/api/todos/1/completed -H "Authorization: Bearer (JWTトークン)"
Resopnse (No Content)

todo削除 DELETE /api/todos/:id

Request curl -X DELETE (IPアドレス):8080/api/todos/1 -H "Authorization: Bearer (JWTトークン)"
Resopnse (No Content)
